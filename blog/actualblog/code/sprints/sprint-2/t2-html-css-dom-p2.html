Reflections

What is grid based design?

The whole page structure is made up of a grid where each element takes the form of one or several boxes. This makes it much simple to structure a page as it accounts for the empty just as much as the filled space. Once the grid has been established, working on the subtleties becomes much more straight-forward.

Whats all the hype about responsive webpages?

A webpage that can adapt efficiently and effectively to different screen types is very attractive in this day and age where people have easy access to devices of all shapes and sizes. Clearly a webpage designed to be seen on a computer will not look very attractive on a mobile phone. If a developer can code a webpage so that the very same page is independantly responsive to the device type and adapts accordingly so the user does not even notice a difference, this is good, responsive design!

What is semantic structure?

How the information is distributed within semantic tags. For example a what sematic tag you would give a heading compared to that which you would give a paragraph. It is important to give data the appropriate semantic tags both for future reference and correct interpretation by the internet browser.
