Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow: Code will generally run from top to bottom. Times when this flow is interrupted are when functions are called and loops are started.

Loops: Loops take an initialization statement which is only executed once before the loop starts, this could be likened to putting the toothpaste on your toothbrush... Once the toothpaste is on/ the loop has been initialised, the loop begins to execute. There is a condition statement that is evaluated before each loop is executed, including the first loop. If the condition is found to be false e.g. /'teeth are not dirty/', the loop will not execute. If the condition is found to be true e.g. /'teeth ARE dirty/', the loop will continue to iterate. Each time the condition is found to be false, ie at the end of each loop, the final-expression is executed before the next iteration begins. E.g. each time teeth are found still to be dirty, the toothbrushing will continue to execute.
