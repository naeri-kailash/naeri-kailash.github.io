Describe all the use cases for each of:

parentheses () - e.g. '1. Execute a function var result = doSomething()'
  • Used for function calls and inputting a variable into a function call \n
  • Surrounding conditional statements
  • Grouping in order to enforce a particular order in which operations are completed \n e.g. x = (2 + 3) * 5
brackets []
  • Mostly used to access the properties inside of an Object or the elements of an array
braces {}
  • Used to enclose blocks of code - e.g. function definitions, conditional blocks, loops
  • Declaration of Object Literals
single quotes '' and double quotes ""
  • You can used them interchangeably. It is particularly when you need to use two sets of quotes on one string/ sentence, then you can use one double and one single so they are differentiated in their uses.
