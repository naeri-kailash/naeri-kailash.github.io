.map() .filter() and .reduce() functions and what they do

  • .map() - this creates a new array with the results of calling a provided function on every element in the original array, eg the function may multiply each element by 2 so the new array will have the multiplied numbers while the original array remains unchanged.
  • .filter() - can filter the contents of an array and output the elements that pass the 'test' which you can input as a function the the brackets.
  • .reduce() - reduces the array to a single value. It executes the provided function for each value of the array (from left-to-right). Subsequently the return value of the function is stored in an accumulator (result/total).
